# Hộp Tải Truyện từ banxia.cc

Tampermonkey script giúp tải toàn bộ truyện từ [https://www.banxia.cc](https://www.banxia.cc) thành file `.txt`.

## 🚀 Tính năng
- Tự động lấy danh sách chương
- Tải về tất cả nội dung chương và lưu thành file `.txt`
- Giao diện đơn giản, nhẹ

## 📥 Cài đặt script

Cài trực tiếp tại:  
[Nhấn để cài](https://github.com/LuongVu2112/banxia-downloader/raw/main/banxia-downloader.user.js)

## 🔧 Hướng dẫn sử dụng
1. Cài Tampermonkey
2. Cài script từ link trên
3. Truy cập trang **mục lục truyện** tại `https://www.banxia.cc/book/ID/`
4. Sử dụng nút tải truyện để lưu về file TXT
